# ESIGEH - Sistem Informasi Geografis Evakuasi Gempa

Aplikasi Web GIS untuk membantu navigasi evakuasi gempa di wilayah **Tanjung dan Pemenang, Lombok Utara**.

## Fitur Utama
- **Geolocation**: Temukan lokasi Anda secara real-time.
- **Nearest Evacuation**: Cari titik evakuasi terdekat secara otomatis.
- **Real-Road Routing**: Petunjuk arah berdasarkan jalan raya asli.
- **Simulasi Lokasi**: Tekan lama pada peta untuk mencari rute dari titik mana saja.
- **Layer Management**: Aktifkan/Matikan layer Batas Administrasi, Pemukiman, dan Label Peta.

## Tech Stack
- **Frontend**: React, Leaflet, Tailwind CSS.
- **Backend**: Node.js, Express.
- **Database**: PostgreSQL with PostGIS extension.
